Push next _config.yml  to config/

常用的命令
- hexo s 启动hexo本地的服务器
- hexo n 新建一篇文章  hexo new "my first article"
- hexo g 生成
- hexo d 部署
- hexo p 发布 Moves a draft post from _drafts to _posts folder.
- hexo clean 清楚所有的缓存
